Summary


Predicting Stock Movement with Performance Data

It can be cost Intensive to the unexperienced trader to constanstly buy and sell with the dynamic changes of the Market. In this portion of the assignment, the target is to focus on price movement per quarter. We want to determine if there will be a positive( > 5%) or negative (< 5%) price movement in the subsequent quarter. If so, the trader can decide to sell, and if not, the trader can decide to hold until there is such a change.

Data Sources used are Quandl.com and Morningstar.com. Quandl’s API is used to retrieve Open, High, Low, Close, Volume data, and Morningstar’s API is used to retrieve Performance data. The data is then cleaned, Analysis is performed to select appropriate features for Machine Leaning model, and a Machine Learning Model is used to train and test data. Finally, accuracy of model is recorded.

KNN: K-Nearest Neighbors Machine Learning Algorithm


Selecting Best K-Value
Breif Description

KNN is a non-parametric, unsupervised learning algorithm that doesn’t make any assumptions on the data distribution. The training phase is very minimal as no generalizations are made in the training phase. The KNN algorithm is based on feature similarity, which is why we utilize a correlation matrix to determine likeness within the features available in the dataset.
