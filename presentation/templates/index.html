Stock Price Analysis

An Introduction to Predicting the Stock Market with Machine Learning

What is this and what our the goal?

This is our final project for our data analytics bootcamp. We will be analyzing stock price data (mostly large-cap U.S. equities) in an attempt to discover which machine learning techniques are best at predicting financial data. To be fair, predicting the stock market is a famously difficult problem, so at a high level this is mostly an exercise in using some of the tools that we've learned throghout the year.

The languages and tools used so far in this project include:

  • HTML
  • CSS
  • JavaScript
    • jQuery
    • Plotly.js
  • Bootstrap
  • Flask
  • Heroku
  • Python
    • BeautifulSoup
    • requests
    • MySQLdb
    • quandl
    • Matplotlib
    • pandas
    • Keras
    • Tensorflow
    • Tweepy
  • MySQL

Ultimately, we decided to compare multivariate machine learning models like K-Nearest Neighbors vs. univariate techniques like Recurrent Neural Networks vs. Natural Language Processing techniques.

Gathering the Data

In this first step, we created a pipeline from our online data provider to our local MySQL database. We used BeautifulSoup and requests to scrape Wikipedia for the ticker symbols of the S&P500 stocks. We then queried our database for these symbols and used them with Quandl's API to download the historical pricing data for each symbol from 1/1/2000 to today. Finally, we queried our MySQL database for relevant stock data and converted it to a pandas DataFrame to use for further analysis.
